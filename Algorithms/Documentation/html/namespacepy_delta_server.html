Detailed Description

A server script written in Python that sends GCode commands to a delta robot.

The commands sent to the robot are determined by selections made on an Android or Web-Based client. The delta bot will have G-code G-code compatible firmware installed on its control board. This script uses Printrun libraries provided by kliment on GitHub.

Author: Dave Mariano Date: Feb 1, 2015 Filename: pyDeltaServer.py Github: github.com/d-mariano/DeltaLaser
